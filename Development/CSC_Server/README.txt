Run server.py file in python windows. You can use IDE like VisualStudio code to run the file. After start up the server, only then run the attendanceSystem.py file. 

Below is the API to query the data:

=================== CSC Face Recognition API ==================
1. /api/query/employees/$<user_id>
2. /api/query/employees
3. /api/query/employees/start=<start_date>
4. /api/query/employees/start=<start_date>&end=<end_date>
5. /api/query/employees/$<user_id>/start=<start_date>
6. /api/query/employees/$<user_id>/start=<start_date>&end=<end_date>
7. /api/query/registeredEmployees
